# 安装
## 下载
在此下载Release版本：https://github.com/kb1000fx/hexo-theme-insulin/releases
或直接点击`Download Zip`下载最新版。

## 修改配置文件
将下载的主题文件解压后放入Hexo目录下的theme文件夹中

修改Hexo目录下的`_config.yml`，把`theme:`的值改为解压后的文件夹名。
